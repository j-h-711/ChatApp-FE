# Project

React.js, Node.js, WebSocket 을 사용한 실시간 단체 채팅 회의 시스템 (반응형)

## 기술스택

- React.js
- Styled-components
- Node.js
- socket.io

## 기능

- 회원가입
- 로그인
- 방 생성 / 삭제
- 방 입장 시스템 메시지
- 대화방 입장 / 퇴장
- 실시간 채팅 (메시지 발신, 수신)
